3- Which actors/actress has played at which year most?¶

  • We want to find the movie counts grouped by actors/actress and years
  • Like genre, we need to split cast data seperated by '|'.
In [117]:
df_cast_list = []
df_cast = []
cast_dict ={}

# split cast data and keep them in a dictionary to count them by year
for i in range(0, len(df)):
    df_cast = df.iloc[i]["cast"].split("|")
    if df_cast[0] == "No cast available": # these records are not appended to the final array
        continue
    df_cast = [x + "-" + str(df.iloc[i]["release_year"]) for x in df_cast]
    df_cast_list = np.append(df_cast_list, df_cast)

# get the count of cast-year pairs
unique, counts = np.unique(df_cast_list, return_counts=True)

# create a dictionary from final tuples
cast_dict = dict(zip(unique, counts))

# the dictionary is reversely sorted to get actors/actress played most at a single year at the beginning
cast_dict = sorted(cast_dict.items(), key=lambda x:x[1], reverse=True)

#print top 5 records
for i in range(0,5):
    print(cast_dict[i])

Conclusions¶

  • We can say that the data is clean in general.
  • We just needed to fill null values with a default values:
    • "cast" column has 76 NULL values. We fill them with "No cast available"
    • "genres" column has 23 NULL values". We fill them with "No genre available"
    • "keywords" column has 1493 NULL values". We fill them with "No keywords available"
    • "production_companies" column has 1030 NULL values". We fill them with "No production company available"
    • "director" column has 44 NULL values". We fill them with "No production company available"
  • Certain columns, like ‘cast’ and ‘genres’, contain multiple values separated by pipe (|) characters. We need to parse them before investigating these values
  • There is no a direct correlation between Science Movies and years. But these movies are popular in 2015 mostly
  • From 2005, number of movies is increasing significantly
  • Runtimes of movies are increasing slightly by years
  • Woody Allen is the directory woh has the most movies with 45 movies.
  • 2014 is the year which movies have been released the most.
  • In 2015 Drama movies were very popular while Western ones is the least one.
  • The actors/actress have maximum 10 movies by year. James Franco has 10 movies in 2013
  • Some additional researchs could be interesting on this data set such as:
    • Percentage of genres by years
    • average budgets per genres
    • average budgets per year
    • Relation between budgets and popularities
